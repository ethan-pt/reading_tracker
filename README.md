  A Django application to track your reading.

## Technologies Used

* Python 3.10.1
* Django
* Google Cloud Platform (GCP)
  * Cloud Run
  * Cloud SQL
  * Cloud Storage Buckets
  * Cloud Secrets
* Docker
* Gunicorn

## Features

* Track number of books read in a year
* Track current page
